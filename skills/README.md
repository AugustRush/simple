# Built-in Skills

Place built-in skill bundles in this directory.

Each skill should live in its own directory and include a `SKILL.md` entrypoint. Supporting files such as templates, examples, and scripts may live alongside it.

Example:

```text
skills/
└── review/
    ├── SKILL.md
    ├── template.md
    ├── examples/
    │   └── sample.md
    └── scripts/
        └── validate.sh
```
